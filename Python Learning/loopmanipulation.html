2.5 Loop Manipulation in Python


1. BREAK

The break statement stops the loop when a condition is met.


Example:

fruits = ["apple", "banana", "cherry"]
for x in fruits:
    print(x)
    if x == "banana":
        break

2. CONTINUE

The continue statement skips an iteration when a condition is met.


Example:

fruits = ["apple", "banana", "cherry"]
for x in fruits:
    if x == "banana":
        continue
    print(x)

3. PASS

The pass statement is used when a statement is syntactically required but you do not want to execute any command.


Example:

for i in range(1, 5):
    if i == 3:
        pass
    print("The current number is", i)

4. ELSE

i) Else with For Loop

The else block executes when the loop finishes normally.


Example:

for x in range(3):
    print(x)
else:
    print("Loop ended normally")

ii) Else with While Loop

The else block executes when a while loop ends.

i = 1
while i < 6:
    print(i)
    i += 1
else:
    print("i is no longer less than 6")
