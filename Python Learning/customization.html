5.6 Customization vs Inheritance


Inheritance uses an attribute definition tree where object.attr searches in an upward direction of the tree for the first occurrence of the word "attr". Lower definitions of the methods in the tree define or override upper definitions of the methods in the inheritance structure.


5.6.1 Specializing Inherited Method

During the execution of the inheritance tree, the subclass name is located prior to the superclass name. The direct superclass method is called using Class.method(self,...).


Example:

class A:
    def display(self):
        print("Super Class")

class B(A):
    def display(self):
        print("Beginning of Sub Class")  
        super().display()
        print("Ending of Sub Class")
