6.2 File Handling


File is a named location on the disk to store information. Files store information permanently on secondary memory, making them a persistent data structure.


6.2.1 Opening a File in Different Modes

All files in Python Programming are required to be open before some operation can be performed on the file. In Python Programming while opening a file, file object is created and by using the file object we can perform different set of operations on the opened file.



Syntax:

file_object = open(file_name, mode)

Example:

file1 = open("d:\\myfile1.txt", 'w')
file2 = open("myfile2.txt", 'rt')

6.2.2 Accessing File Contents

For reading the file, we must open the file in r mode and we must specify the correct name of the file which has to be read. The read() method is used for reading the file.


Example:

file1 = open("d:\\myfile1.txt", 'w')
file2 = open("myfile2.txt", 'rt')

The readline() method :

The readline() method allows us to read a single line from the file. When file reaches to the end, it returns an empty String


Example:

                
file=open("d:\\myfile.txt",'rt')
print(file2.readline())

The readlines() method:

The readlines() method is used to print all the lines in the program.


Example:

file=open("d:\\myfile.txt",'rt')
print(file2.readlines())

6.2.3 Writing to a File

For writing the contents to the file we must open it in writing mode. hence we use ‘w’ or ‘a’ or ‘x’ as a file mode. the write() method is used to write the contents to the file.


Example:

file = open("d:\\myfile.txt", 'wt')
file.write("Hello Everyone")
file.close()

The writelines() method:

The writelines() method is used to write list of strings to the file.


Example:

file=open("d:\\myfile.txt",'wt')
line=["This is the Line that we want to write in file"]
file.writelines(line)

Appending the File :

Appending the file means inserting records at the end of the file. For Appending the file we must open the file in ‘a’ or’ab’ mode.


Example:

file=open("d:\\myfile.txt",'a')
file.write("Lets write in file")

6.2.4 Closing a File

After performing all the file operations, it is necessary to close the file. Closing the file is necessary because it frees all the resources that are associated with the file.


Syntax:

file_object.close()

Example:

file = open("d:\\myfile.txt", 'rt')
    file.close()

6.2.5 Renaming a File

For renaming a file in Python, the rename() method is used. For that purpose it is required to import the module os.



Syntax:

os.rename( src , dst )

Example:

import os
os.rename(r'D:\file1.txt',r'D:\file2.txt',)
print("File Renamed Successfully")

6.2.6 Deleting a File

For Removing a file in Python, the remove() method is used belonging to the os module.


Syntax:

os.remove(filename)


Example:

import os
os.remove('D:\\file1.txt')
print("File Removed Successfully")

6.2.7 Directories in Pyth

Seek and Tell Method

i)The Seek Method:

The seek method is used to change the file position. Similarly, the tell method returns the current position. The method seek() sets the file's current position at the offset.


Syntax:

fileObject.seek(offset[, whence])

ii)The Tell Method:

The method tell() returns the current position of the file read/write pointer within the file.


Syntax:

fileObject.tell()

Example:

file = open('d:\\test.txt', 'rt')
print(file.read())
print(file.tell())
file.seek(0)
print(file.tell())

Directory Methods


1.Getting Current Working Directory

For getting the name of current working directory, we use the function name getcwd() method. This method returns the name of current working directory.


Example:

import os
print(os.getcwd())

2.Listing Files in a Directory

For listing the contents of a directory, the listdir() method is used.


Example:

import os
print(os.listdir())

3.Creating and Removing Directories

A new directory can be created using the mkdir() method.

A file can be deleted using the remove() method. The rmdir() method removes an empty directory.


Example:

import os
os.mkdir("mypythonPrograms")
os.rmdir("mypythonPrograms")
