6.3 Exception Handling


An exception is an event that occurs during the execution of a program that disrupts the normal flow of execution. If not handled properly, it may terminate the program.


Types of Errors

  • Compile Time Errors: Syntax errors like missing colons or incorrect indentation.
  • Run Time Errors: Errors that occur due to invalid input or unexpected behavior.
  • Logical Errors: Errors due to incorrect program logic.


Example:


print(Hello)  # Syntax Error (Missing quotes)

a = 10
if a > 5  # Syntax Error (Missing colon)

6.3.1 try: except Statement

The try...except block is used to handle exceptions in Python. The code that may cause an exception is placed inside the try block, while the except block handles specific exceptions.


Syntax:


try:
    # Suspicious code
except Exception1:
    # Handle Exception1
except Exception2:
    # Handle Exception2
else:
    # Execute if no exception occurs

Example:

a = int(input("Enter value of a:"))
    b = int(input("Enter value of b:"))
    try:
        c = a / b
    except ValueError:
        print("Wrong data entered")
    except ZeroDivisionError:
        print("Divide By Zero")
    else:
        print("The result is", c)

6.3.2 The Raise Statement

Raising an exception is a technique for interrupting the normal flow of execution in a program, signaling that some exceptional circumstance has arisen, and returning directly to an enclosing part of the program that was designated to react to that circumstance. The Python interpreter raises an exception each time it detects an error in an expression or statement.


Syntax:

raise [Exception [, args [, traceback]]]


Example:


def NegativeNum(num):
    if num < 0:
        raise Exception("Number is negative!", num)

6.3.3 Standard Built-in Exceptions

Python provides several built-in exceptions like TypeError, ValueError, ZeroDivisionError, etc.


6.3.4 User Defined Exceptions

Python allows users to create their own exceptions by defining custom exception classes.


Example:

class InvalidAgeError(Exception):
    def __init__(self, age, msg="Age must be between 0 and 120"):
        self.age = age
        self.msg = msg
        super().__init__(self.msg)

    def __str__(self):
        return f'{self.age} -> {self.msg}'

def set_age(age):
    if age < 0 or age > 120:
        raise InvalidAgeError(age)
    else:
        print(f"Age set to: {age}")

try:
    set_age(150)
except InvalidAgeError as e:
    print(e)
