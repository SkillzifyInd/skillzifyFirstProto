5.5 Inheritance and Composition Classes


Inheritance

Inheritance is a relationship similar to a family tree where the properties of one class are inherited by another class. It provides core reusability.


Types of Inheritance

  • Single Inheritance
  • Multiple Inheritance

Single Inheritance

In single inheritance, there is one base class and one derived class.



Syntax:


CLASS A :
#PROPERTIES OF CLASS A
                    
CLASS B (A):
#CLASS B INHERITING THE PROPERTIES OF CLASS

Example:

class Person:
    def display(self):
        print("I am Tanvi")

class Student(Person):
    def show(self):
        print("I am in Final Year")

s = Student()
s.show()
s.display()

Multiple Inheritance

In multiple inheritance, more than one child class can be derived from a single base class.



Syntax:


Class Base1:
     Body of the class
Class Base2:
    # Body of the class
Class Derived(Base1, Base2):
    # Body of the class

Example:

class Class1:
    def m(self):
        print("First Year")

class Class2(Class1):
    def m(self):
        print("Second Year")
 
class Class3(Class1):
    def m(self):
        print("Third Year")  
       
class Class4(Class2, Class3):
    pass  
     
obj = Class4()
obj.m()

Composition

Composition is a relationship where one object is defined as a part of another object, rather than using inheritance.


Syntax:

class A :

    # variables of class A
    # methods of class A
              
    ...
                
    class B : 
              
    # by using "obj" we can access member's of class A.
    obj = A()
    # variables of class B
    # methods of class B
              
    ...

Example:

class Component:
    def __init__(self):
        print('Component class object created...')
    
    def m1(self):
        print('Component class m1() method executed...')

class Composite:
    def __init__(self):
        self.obj1 = Component()
        print('Composite class object also created...')
    
    def m2(self):
        print('Composite class m2() method executed...')
        self.obj1.m1()

obj2 = Composite()
obj2.m2()
