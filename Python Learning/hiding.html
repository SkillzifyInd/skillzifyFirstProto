5.3 Data Hiding


Data Hiding is a mechanism which does not allow to access attributes outside the class. Data Hiding is an important property of object oriented Programming.

we need to name attribute with double underscore(__) prefix and those attribute then are not be directly visible to outsiders. Any variable prefix with double underscore is called as private variable which is accessible only with the class where it is declared.


Example:

class Counter:
    _secretCount = 0  # Private variable
    def count(self):  # Public method
        self.__secretCount += 1
        print("count=", self.__secretCount)  # Accessible inside the class

c1 = Counter()
c1.count()  # Invoke method
c1.count()
try:
    print("Total count=", c1.__secretCount)  # Cannot access private variable directly
except AttributeError as e:
    print(f"AttributeError: {e}")
