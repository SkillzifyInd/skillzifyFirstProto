4.4 Python Packages

A package is a hierarchical file directory structure that defines a single Python application environment that consists of modules, subpackages, and deeper nested sub-subpackages. Along with these, a package contains a file named __init__.py.



Key Components of a Python Package

  • Module: A single Python file containing reusable code (e.g., math.py).
  • Package: A directory containing modules and a special __init__.py file.
  • Sub-Packages: Packages nested within other packages for deeper organization.

4.4.1 Writing Python Package

Packages allow for a hierarchical structuring of the module namespace using dot notation.

Syntax 1: Importing Entire Package


Example:

import pkg.mod1, pkg.mod2
pkg.mod1.m1()  # Output: first module

Syntax 2: Importing Specific Function


Example:

from pkg.mod1 import m1
m1()  # Output: first module

Syntax 3: Importing with Alias


Example:

from pkg.mod1 import m1 as module
module()  # Output: first module

4.4.2 Using Standard Package


1. What is NumPy?

NumPy is the core library for scientific computing in Python. It provides a high-performance multidimensional array object and tools for working with these arrays.

Advantages:

  • Array-oriented computing.
  • Efficiently implemented multi-dimensional arrays.
  • Designed for scientific computation.

Example:

import numpy as np
arr = [0,2,4,6,8,10]
even_arr = np.array(arr)
print(even_arr)  # Output: [0 2 4 6 8 10]

2. What is SciPy?

SciPy is an open-source Python library used for mathematics, scientific computing, and engineering. It is built on NumPy and contains many useful functions.


Example:

from scipy.special import cbrt
cb = cbrt([27,125])
print(cb)  # Output: [3. 5.]

3. What is Matplotlib?

Matplotlib is a plotting library that provides functions for various types of graphs and visualizations.


Example:

import numpy as np
import matplotlib.pylab as plt

arr = [5,10,15,20,25]
a = np.array(arr)

plt.plot(a)
plt.show()

4. What is Pandas?

Pandas is a Python package designed for efficient data manipulation and analysis. It provides data structures such as Series and DataFrame.


Example:

import pandas as pd
import numpy as np

a = np.array(['A', 'B', 'C', 'D'])
s = pd.Series(a)
print(s)
