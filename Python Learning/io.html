6.1 I/O Operations

In any programming language, an interface plays a very important role. It takes data from the user (input) and displays the output. One of the essential operations performed in Python is to provide input value to the program and output the data produced by the program to the standard output device (monitor). Some functions like input() and print() are widely used for standard Input and Output operations, respectively.


6.1.1 Reading Keyboard Input

Python provides a built-in function to read a line of text from standard input, which by default comes from the keyboard. For that purpose, the function input() is used.


Syntax:

input([prompt])

Where prompt is the string we wish to display on the screen.


Example:

name = input("Enter a Name: ")
print("Name is", name)

6.1.2 Printing to Screen

The function print() is used to output data to the standard output device, i.e., monitor/screen. The output can be redirected or stored into a file as well. Using the print function, we can print information to the screen. The message can be a string or another object, which will be converted into a string before being displayed.


Syntax:

print("Message to be printed")

Example:

print("Good Morning")
