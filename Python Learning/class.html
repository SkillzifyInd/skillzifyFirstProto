5.1 Python Classes and Objects


Classes

Class is a basic entity in object oriented programming. In any object oriented programming, the class is declared using keyword “class”. Every class consist of data members and methods.


Example:

class Student:
    roll_no = 33
    name = "Tanvi"
obj = Student()
print("Roll no ", obj.roll_no)
print("Name is ", obj.name)

Objects

An object is an instance of a class that has some attributes and behavior. Objects are runtime entities of the object oriented programs.


Example:

class Student:
    roll_no = 33
    name = "Tanvi"
obj = Student()
print("Roll no ", obj.roll_no)
print("Name is ", obj.name)

How To Create a Class and Object ?

The keyword ‘ class ‘ is used to define the class followed by the class name . Objects can be used to access the attributes of the class.


Syntax of Class:

Class class_name :
Statement1
Statement2
 ….
Statementn

Syntax of Object:

Object_name=class_name()

Example:

class Student :
    roll_no=33
    name="Tanvi"
obj=Student()
print("Roll no ",obj.roll_no)
print("Name is ",obj.name)

Class Methods and Self Object

The class methods are the function define inside the class. They are actually the normal functions but belong to the class. The class method defines the first argument as self. The self methods are accessible from outside the class using object. Hence these methods are called as instance methods.


Example:

class Student:
    roll_no = 33
    name = 'Tanvi'
    def show(self):
        print("Inside the Student Class")
obj = Student()
print("Roll Number: ", obj.roll_no)
print("Name: ", obj.name)
obj.show()

Public and Private Members

Public Variables

Public variables are those variables that are defined in the class and can be accessed from within the class and from outside the class.


Example:

class Student:
    def __init__(self, R, N):
        self.Roll = R
        self.Name = N
obj = Student(33, "Tanvi")
print("Roll No: ", obj.Roll)
print("Name: ", obj.Name)

Private Variables

Private variables are those variables that are defined in the class but can be accessible by the methods of that class only.


Example:

class Student:
    def __init__(self, R, N):
        self.__Roll = R
        self.__Name = N
    def get_details(self):
        print("Roll No:", self.__Roll)
        print("Name:", self.__Name)
obj = Student(33, "Tanvi")
obj.get_details()
