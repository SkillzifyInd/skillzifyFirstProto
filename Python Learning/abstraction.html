5.4 Data Abstraction


Data abstraction is a mechanism in which an abstract class is created. An abstract class is a class that contains at least one abstract method, which does not have an implementation. The implementation of these abstract methods is provided by child classes.

Data abstraction helps in providing only essential details about data to the outside world while hiding the implementation details.


Example:

from abc import ABC, abstractmethod

class Color(ABC):  
    @abstractmethod
    def showColor(self):
        pass  

class Red(Color):
    def showColor(self):
        print("This is a red color")

class Green(Color):
    def showColor(self):
        print("This is a green color")

def main():
    r = Red()
    r.showColor()
    
    g = Green()
    g.showColor()

if __name__ == "__main__":  
    main()
