2.2 Control Flow



The program control flow is the order is the order in which program code executes. Python programming provides a control structure which transfers the control from one part of program to some other part of program.


Note : The Control Flow is refer to statement sequencing in a program to get desire result.



Control Flow Statements are of three statements:

  • Conditional Statements
  • Loop Statements
  • Loop Manipulation Statements
