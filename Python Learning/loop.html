2.4 Looping in Python


1. While Loop

The while loop is the most basic looping structure in Python. It executes a sequence of statements repeatedly as long as the condition is true.


Example:

i = 1
while i < 6:
    print(i)
    i += 1

While...Else:

The while loop can also have an else statement, which executes when the condition becomes false.


Example:

i = 1
while i <= 10:
    i += 1
else:
    print("Invalid value of i")

2. For Loop

The for loop iterates through a sequence (list, tuple, dictionary, string, etc.), executing the block of code for each item in the sequence.


Example:

fruits = ["apple", "banana", "cherry"]
for x in fruits:
    print(x)

3. Nested Loop

Nested loops are loop structures where one loop is inside another loop. For example, we can use nested for loops.


Example:

adj = ["red", "big", "tasty"]
fruits = ["apple", "banana", "cherry"]

for x in adj:
    for y in fruits:
        print(x, y)
