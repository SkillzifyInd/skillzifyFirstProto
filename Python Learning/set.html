3.3 Python Sets


Sets data structure in Python programming is implemented to support mathematical set operations . Sets are based on specific rules they define .


1. Creating a Set

Sets store multiple unique items in a single variable, using curly braces or the set() constructor.


Example:

Emp = { "Tanvi" , 9 , 'T' }
a = set('abc')
print(Emp, a)

2. Accessing a Set

We cannot access the values in the set using index as the set is unorder and has no index .


Example:

a = {1, 2, 3, 4}
print(a)

3. Deleting Values in a Set

For removing the item from the set either remove() method or discard() method is used .


i) Using remove()


Example:

a = {1, 2, 3, 4}
a.remove(2)
print(a)

ii) Using discard()


Example:

a = {1, 2, 3, 4}
a.discard(3)
print(a)

4. Updating a Set


i) Using update()

Once the set is created we cannot change the values in set but we can add the element to the set we can update the set by using the add() method or update() method .


Example:

a = {1, 2, 3, 4}
a.update([5, 7])
print(a)

ii) Using add()


Example:

b = {1, 2, 3, 4}
b.add(8)
print(b)

Basic Set Operations


i) Union

Union operation perform on two sets return all the elements from both the sets .


Example:

x = {'a', 'b', 'c'}
y = {'d', 'e'}
z = x.union(y)
print(z)

ii) Intersection

Intersection operation perform on two sets Returns all the elements which are common or in both the sets .


Example:

x = {'a', 'b', 'c'}
y = {'a', 'c', 'd', 'e'}
z = x & y
print(z)

iii) Difference

Difference operation on two sets set 1 and set to Returns all the elements which are present on Set 1 but not in Set 2 . it is performed by using - operator .


Example:

x = {'a', 'b', 'c'}
y = {'a', 'c', 'd', 'e'}
z = x - y
print(z)

iv) Symmetric Difference

The symmetric represent set of all elements which belong to either to A or B but not both . Difference operation is performed by using ^ operator


Example:

x = {'a', 'b', 'c'}
y = {'a', 'c', 'd', 'e'}
z = x ^ y
print(z)

Built-in Set Functions


i) all()

This functions return true if all yments of the set are true this function also returns a true if the value in the set is empty


Example:

print(all({1, 2, 3}))  # True
print(all({0, 1, 2}))  # False

ii) any()

This function return true if any element of the set is true


Example:

print(any({0, 0, 1}))  # True

iii) len()

len() function returns the length of the set


Example:

x = {1, 2, 3, 4}
print(len(x))

iv) max() and min()

This function returns the maximum value and minimum value from the set .


Example:

x = {5, 10, 15}
print(max(x))
print(min(x))

v) sorted()

This function returns a new sorted list from the elements .


Example:

x = {3, 1, 4, 2}
print(sorted(x))

vi) sum()

This function returns the sum of all elements in the set .


Example:

x = {1, 2, 3, 4}
print(sum(x))
