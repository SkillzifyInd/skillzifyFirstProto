5.2 Method Overloading and Overriding


5.2.1 Method Overloading

Method overloading is an ability of a function to behave differently based on the number of parameters passed to it. In method overloading, the same function name is used for different behaviors.


Example:

def area(length, breadth):
    cal = length * breadth
    print(cal)

def area(size):
    cal = size * size
    print(cal)

area(3)
area(4, 5)

5.2.2 Method Overriding

Method overriding is a mechanism in object-oriented programming that allows a child class to change the implementation of a function defined in its parent class.


Criteria for Method Overriding:

  • There must be inheritance between the classes.
  • The function redefined in the child class must have the same number of arguments as in the parent class.

Example:

class A:
    # Parent class
    "Parent Class"
    def display(self):
        print('This is base class.')

class B(A):
    # Derived class
    "Child/Derived class"
    def display(self):
        print('This is derived class.')

obj = B() # Instance
obj.display() # Calls child class method
