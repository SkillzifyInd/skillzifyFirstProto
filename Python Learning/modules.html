4.3 Modules

Modules are Python (.py) files that contain code such as functions, classes, and variables. These modules can be imported into scripts for reuse.


4.3.1 Writing Modules

Writing a module means simply creating a file which contains python definitions and statements.

The file name is the module name with the extension .py . To include module in a file, use import statement.


Example:

# mymodule.py
def greeting(name):
    print("Hello, " + name)

4.3.2 Importing Objects from Modules

Import statement is used to import a specific module by using its name

After using import statement we can refer the things defined in that module.


Example:

from mymodule import greeting
greeting("Tanvi")

4.3.3 Name of the Module

Module has a name. one can use the name of the module using _name_ attribute of the module of the module


Example:

print("Name of Module is", __name__)

4.3.4 Python Built-in Modules


Math Module

The Math module used in Python for various mathematical functionalities such as finding log computing sign and cosine values calculating square root and so on .


Example:

import math
print(math.sqrt(25))
print(math.factorial(5))
print(math.pow(2, 3))
print(math.pi)

Functional Programming Modules

The functional programming module provides functions and classes that support functional programming style.


Example:

import itertools
num = [1,2,3,4,5]
print(list(itertools.combinations(num, 3)))

4.3.5 Namespace and Scoping

Namespaces ensure that variable names do not conflict. Each module has its own namespace.


Example:

# FirstModule.py
def display(a, b):
    return a + b

# SecondModule.py
def display(a, b):
    return a * b
            
import FirstModule
import SecondModule
print(FirstModule.display(10, 20))
print(SecondModule.display(5, 4))

4.3.6 Global, Local, and Built-in Namespace


  • Global Namespace: Contains module-level variables and functions.
  • Local Namespace: Exists within functions.
  • Built-in Namespace: Includes built-in functions like print(), len().
