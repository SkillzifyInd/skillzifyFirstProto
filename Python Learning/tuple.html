3.2 Python Tuple


Tuple is a collection of element which are enclosed within the parenthesis and this elements are separated by commaâ€™s .


1. Creating a Tuple

Tuples store multiple items in a single variable, using parentheses.



Example:

            
a = ("AAA", "BBB", "CCC")
b = (10, 20, 30, 40)
print(a, b)

2. Accessing Values in a Tuple

Accessing items or elements from the tuple is a method to get values stored in the tuple from a particular location on index. The elements in tuple can be accessed by using the index of the element.


Example:

            
a = ("apple", "banana", "cherry")
print(a[1])

3. Deleting a Tuple

Tuples are unchangeable , so we cannot remove items from it but we can delete the tubel completely .We can delete the tuple by using the del statement .


Example:

            
a = ("apple", "banana", "cherry")
print(a)
del a

4. Updating Values in Tuple

Tuples are immutable , this means values in the tuple cannot be changed . we can only extract values of one Tuple to create another Tuple .


Example:

            
a = ("A", "B", "C")
b = ("D", "E")
c = a + b
print(c)

Basic Tuple Operations

i) Finding Length

The length function is used to find the number of elements in the tuple .


Example:

a = ("apple", "banana", "cherry")
x = len(a)
print(x)

ii) Repetition

For repetition of the elements in the tuple , * operator is used.


Example:

a = (1, 2, 3)
print(a * 3)

iii) Membership

Membership operation is used to check whether the element is present in the tuple or not . Membership operation is performed by using in operator .


Example:

a = (1, 2, 3)
print(4 in a)

iv) Iteration

Iteration means accessing the individual elements in the Tuple. Elements in the tuple can be accessed by using the for loop .


Example:

a = (1, 2, 3)  
for i in a:    
    print(i)

Built-in Tuple Functions

i) max()

This function returns the maximum value from the tuple a .


Example:

a = (10, 20, 30, 40)
print(max(a))

ii) min()

This function returns minimum value from the tuple a .


Example:

a = (10, 20, 30, 40)
print(min(a))

iii) tuple(sequence)

This function converts the list into tuple


Example:

list1 = ["apple", "banana", "cherry"]
tuple1 = tuple(list1)
print(tuple1)
